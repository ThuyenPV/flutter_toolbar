# 💻 Installation

In the `dependencies:` section of your `pubspec.yaml`, add the following line:

```yaml
dependencies:
  flutter_toolbar: <latest version>
```

# ❔ Usage

### Import this class

```dart
import 'package:flutter_toolbar/flutter_toolbar.dart';
```

### Flutter ToolBar

```dart
    FlutterToolbar(
        toolbarItems: [
            ToolbarItemData(
                title: 'Edit',
                color: Colors.pinkAccent,
                icon: Icons.edit,
            )
        ],
    ),
```
